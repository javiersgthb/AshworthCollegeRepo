Lesson 4: Introduction to Visual C#

In this lesson, you will remove all Visual elements of a graphical user interface to expose the principles of programming with C# via a console application in Visual Studio. You will write simple C# apps using code rather than visual programming; write decision-making statements; and use relational and equality operators.

Learning Objectives

  • Objective 1 Create a simple C# console application and later make changes to that program.
  • Objective 2 Analyze the basic syntax of a class and variable declaration.
  • Objective 3 Explain the concept of keywords as used in C#.
  • Objective 4 Explain the correct use of arithmetic and relational operators.
  • Objective 5 Write decision-making statements.
