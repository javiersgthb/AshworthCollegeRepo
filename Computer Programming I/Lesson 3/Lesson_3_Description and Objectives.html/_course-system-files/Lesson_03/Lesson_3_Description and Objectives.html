Lesson 3: The Visual Studio Integrated Development Environment (IDE)

In this lesson, you will learn about the Visual Studio 2012 programming environment for creating and testing application code using the Visual C# (pronounced Cee Sharp) computer programming language. You will start with the basics of the Integrated Development Environment (IDE), the application that allows the programmer to write programming code, and then have the compiler convert that code into machine code that the computer CPU requires. You will also learn that there are two main things that make up a Graphical User Interface (GUI): The form (the window you see on the screen), and controls, such as text boxes, check boxes, and buttons.

Learning Objectives

  • Objective 1 Describe the basic components of Visual Studio Express 2012.
  • Objective 2 Demonstrate how to use the Help function and to invoke context-sensitive help features.
  • Objective 3 Describe the key commands contained in the IDE menus and toolbars.
  • Objective 4 Explain what visual app development is and how it simplifies and speeds app development.
  • Objective 5 Use visual app development to create, compile, and execute a simple Visual C# application.
