Lesson 7: Graphical User Interfaces with Windows Forms: Part II

This lesson will expose you to a greater variety of visual controls to create more functional and elaborate Windows Forms applications. This lesson will also discuss the creation and maintenance of multiform applications and present custom controls, which you can use to create and reuse a control with particular functionality in other Windows Forms applications.

Learning Objectives

  • Objective 1 Correctly create menus, tabbed windows, and multiple document interface (MDI) programs.
  • Objective 2 Properly use the ListView and TreeView controls to display information.
  • Objective 3 Create hyperlinks using the LinkLabel control.
  • Objective 4  Demonstrate the display of lists of information in ListBox, CheckedListBox, and ComboBox controls.
  • Objective 5 Input dates and times with the MonthCalendar and DateTimePicker controls.
