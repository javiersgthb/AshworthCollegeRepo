Lesson 5: Introduction to Classes, Objects, Methods, and Strings

In this lesson, you will learn the basics of object-oriented programming (OOP), including how to create classes and instantiate objects. You will also explore properties and methods as well as how to use the Unified Modeling Language to diagram an OOP design.

Learning Objectives

  • Objective 1 Declare a class and instantiate an object from that class.
  • Objective 2 Define an object's properties.
  • Objective 3 Correctly use methods to implement an object's behavior.
  • Objective 4 Compare the use of local variables and instance variables in a method.
  • Objective 5 Describe string data types and how to populate an object with data during instantiation.
