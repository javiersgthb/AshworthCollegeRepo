Lesson 6: Graphical User Interfaces with Windows Forms: Part I

This lesson showcases the basic functions and interactivity visual programming provides. You will learn how to create a Windows Forms project, edit form information, add controls to a form, and handle form events generated from the controls.

Learning Objectives

  • Objective 1 Describe the basic elements of and how to create a graphical user interface (GUI).
  • Objective 2 Demonstrate how to process events in response to user interactions with GUI controls.
  • Objective 3 Identify the namespaces that contain the classes for GUI controls and event handling.
  • Objective 4 Demonstrate how to create and manipulate form controls.
  • Objective 5 Demonstrate how to process mouse and keyboard events.
