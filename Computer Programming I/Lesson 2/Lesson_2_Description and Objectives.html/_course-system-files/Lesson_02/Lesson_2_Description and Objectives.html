Lesson 2: Introduction to Visual C#

In this lesson, you will learn about computer hardware and modern computing techniques. You will explore the types of programming available using the C# language. You will also be introduced to some of the basic terminology associated with the Internet, object-oriented programming. You will be introduced to Windows Azure and cloud computing basics.

Learning Objectives

  • Objective 1 Define and describe basic hardware, software, and data concepts.
  • Objective 2 Discuss the different types of computer programming languages.
  • Objective 3 Provide a brief history of the Visual C# programming language and the Windows Operating System.
  • Objective 4 Provide a brief history of the Internet and the World Wide Web.
  • Objective 5 Describe the parts that Windows 10, .NET 4.6, Visual Studio 2015 and C# play in the C# ecosystem.
