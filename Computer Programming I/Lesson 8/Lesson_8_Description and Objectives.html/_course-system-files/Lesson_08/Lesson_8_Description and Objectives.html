Lesson 8: Classes and Objects: A Deeper Look

In this lesson, we take a deeper look at building classes, controlling access to members of a class, and creating constructors. We also discuss composition—a capability that allows a class to have references to objects of other classes as members. The lesson also covers static class members and readonly instance variables in detail. Further, we investigate issues such as software reusability, data abstraction, and encapsulation.

Learning Objectives

  • Objective 1 Explain data structure, encapsulation, and data hiding.
  • Objective 2 Successfully use composition to allow a class to have references to objects of other classes as members.
  • Objective 3 Demonstrate the ability to use keyword this, static variables and methods, and readonly fields.
  • Objective 4 Explain how to throw an exception to indicate that an argument is out of range.
  • Objective 5 Explain how to use object initializers to create an object and initialize it in the same statement.
